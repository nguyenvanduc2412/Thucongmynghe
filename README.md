# thucongmynghe-website

## Overview
This project is a web application for "Bán đồ gỗ thủ công mỹ nghệ Nguyễn Đức", showcasing handcrafted wooden products. It consists of a frontend built with React and TypeScript, and a backend using Node.js.

## Project Structure
The project is organized into two main directories: `frontend` and `backend`.

### Frontend
The frontend is developed using React and TypeScript. It includes the following key components and pages:

- **Components**:
  - `Header`: Renders the website's header with logo and navigation links.
  - `Footer`: Displays footer information such as copyright and contact details.
  - `ProductCard`: Displays product details in a card format.
  - `Navigation`: Provides navigation links to different pages.

- **Pages**:
  - `Home`: The landing page showcasing featured products.
  - `Category`: Displays a list of product categories.
  - `ProductDetail`: Shows detailed information about a specific product.
  - `About`: Provides information about the brand and its values.
  - `Contact`: Includes a contact form for inquiries.

- **Styles**:
  - `global.css`: Contains global styles for the application.
  - `theme.css`: Contains theme-specific styles for a luxurious and classic design.

### Backend
The backend is built using Node.js and handles API requests. It includes:

- **Routes**:
  - `products`: Manages product-related requests.
  - `contact`: Handles contact form submissions.

- **Controllers**: Contains logic for handling requests.

## Setup Instructions

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm start
   ```

### Backend
1. Navigate to the `backend` directory.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm run start
   ```
